# Personal Engineering Portfolio

This repository is the long-term home for a public-facing engineering portfolio. It will document selected tools, prototypes, and technical projects through clear, public-safe project summaries and supporting visual material.

## Featured projects

### Global RF Spectrum Dashboard

A planned dashboard concept for exploring spectrum information through an accessible, map- and data-oriented interface. The portfolio entry will focus on the problem space, interaction design, and implementation approach using public-safe example data.

### Compliance Plan Generator

A planned tool concept for structuring certification and compliance planning workflows. The portfolio entry will describe the user journey, configurable inputs, and generated planning outputs without exposing proprietary rules, datasets, or internal processes.

## Planned technology

The website technology stack has not yet been selected. This repository is intentionally framework-neutral during its initial setup. A future implementation may include a modern frontend framework, static-site or server-rendering tooling, accessible UI components, automated quality checks, and a deployment workflow.

## Development status

Phase 1 repository setup is in progress. No frontend application, framework, or production website has been implemented yet.

## Repository layout

- `website/` — reserved for the future portfolio website.
- `projects/` — public-safe documentation for featured projects.
- `assets/` — images, icons, and screenshots intended for public use.
- `docs/` — planning and supporting documentation.
